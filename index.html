FAQ

Does it work offline?

Yes. Core models run locally; you can download language packs via the app.

Is my data private?

By default, no audio or sign data leaves the glove. Cloud sync is optional and encrypted.

Which sign languages are supported?

ASL and ISL is supported by default. Additional packs available via app.
